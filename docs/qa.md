# Вопросы и уточнения — TellMeApp

## Критические (блокируют разработку)

### Q1. API-ключ AquaVoice
> Какой конкретно API-ключ использовать для тестирования?

**Ответ:** Вводится вручную в Настройках. Ключ получается на https://app.aquavoice.com/api-dashboard
**Статус:** Решено

---

### Q2. Формат аудио для API
> Какой формат аудио оптимален для AquaVoice API?

**Ответ:** WAV (16kHz, 16-bit, mono) — нативный формат AudioRecord
**Статус:** Решено

---

### Q3. Подписка — модель данных
> Какие поля должны быть в карточке подписки?

**Ответ:** Статус, дата окончания, тип тарифа
**Статус:** Решено

---

## Высокий приоритет

### Q4. Формат ссылки активации
> Как работает активация по ссылке?

**Ответ:** Пользователь вставляет ссылку-ключ. Пока валидна — приложение работает.
**Статус:** Решено

---

### Q5. Язык распознавания
> Какой язык распознавания по умолчанию?

**Ответ:** Русский
**Статус:** Решено

---

### Q6. Поведение при ошибке распознавания
> Что делать, если API вернул ошибку или пустой текст?

**Ответ:** Toast "Не удалось распознать" + код ошибки в логах
**Статус:** Решено

---

### Q7. Режим стриминга
> Использовать стриминг (SSE) или batch?

**Ответ:** Batch (non-streaming). Достаточно для вставки текста целиком.
**Статус:** Решено

---

### Q15. AI провайдер по умолчанию
> Какой AI провайдер использовать по умолчанию?

**Ответ:** z.ai. Пользователь может переключиться на Claude на главном экране.
**Статус:** Решено

---

### Q16. Claude модель
> Какую модель Claude использовать?

**Ответ:** `claude-sonnet-4-20250514` — актуальная модель на момент реализации.
**Статус:** Решено

---

### Q17. z.ai URL для Coding Plan
> Какой URL использовать для GLM Coding Plan?

**Ответ:** `https://api.z.ai/api/coding/paas/v4` (не `/api/paas/v4`)
**Статус:** Решено

---

## Средний приоритет

### Q8. Telegram бот
> Логика взаимодействия с Telegram-ботом?

**Ответ:** Разрабатывается в отдельном сервисе. В приложении — только поле для вставки ссылки-ключа.
**Статус:** Закрыт

---

### Q9. Иконка приложения и брендинг
> Есть ли готовый логотип / иконка / цветовая схема?

**Ответ:** Пока нет. Использовать стиль Happ (тёмно-синий фон, голубые акценты).
**Статус:** Решено

---

### Q10. Мультиязычность UI
> Поддерживать несколько языков?

**Ответ:** Русский и английский
**Статус:** Решено

---

### Q11. Package name
> Текущий package: `com.example.tellmeapp`. Оставить?

**Ответ:** `com.TellMeUp.tellmeapp`
**Статус:** Решено

---

### Q18. AccessibilityService на Xiaomi
> Сервис показывает "срабатывает некорректно" на MIUI

**Ответ:** Известная проблема MIUI. Требуется: включить автозапуск, отключить оптимизацию батареи, разрешить работу в фоне. Fallback — clipboard.
**Статус:** Частично решено (fallback работает)
