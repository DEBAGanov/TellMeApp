# Вопросы и уточнения — TellMeApp

## Критические (блокируют разработку)

### Q1. API-ключ AquaVoice
> Какой конкретно API-ключ использовать для тестирования?

**Ответ:** Будет предоставлен позже. Ключ получается на https://app.aquavoice.com/api-dashboard
**Статус:** Ожидает предоставления ключа

---

### Q2. Формат аудио для API
> Какой формат аудио оптимален для AquaVoice API?

**Ответ:** WAV (16kHz, 16-bit, mono) — нативный формат AudioRecord, не требует перекодирования
**Статус:** Решено

---

### Q3. Подписка — модель данных
> Какие поля должны быть в карточке подписки?

**Ответ:**
- Статус (активна / неактивна)
- Дата окончания
- Тип тарифа
**Статус:** Решено (упрощённая модель для MVP)

---

## Высокий приоритет

### Q4. Формат ссылки активации
> Как работает активация по ссылке?

**Ответ:** Пользователь вставляет ссылку-ключ в приложение. Пока ссылка валидна — приложение работает. Когда срок истекает — приложение блокируется и просит продлить (оплата через Telegram бот).
**Статус:** Решено

---

### Q5. Язык распознавания
> Какой язык распознавания по умолчанию?

**Ответ:** Зафиксирован русский
**Статус:** Решено

---

### Q6. Поведение при ошибке распознавания
> Что делать, если API вернул ошибку или пустой текст?

**Ответ:** Toast "Не удалось распознать" + код ошибки
**Статус:** Решено

---

### Q7. Режим стриминга
> Использовать стриминг (SSE) или batch?

**Ответ:** Использовать стриминг (SSE)
**Статус:** Решено

---
